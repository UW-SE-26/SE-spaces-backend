## Getting Started

1. Git clone the repo
2. Install [Node](https://nodejs.org/en/) and [Yarn](https://classic.yarnpkg.com/en/docs/install/)
3. `yarn` to install packages
4. Create a `.env` file
5. `yarn start` to run the project
